# Multi-Agent Reinforcement Learning for UAV-Truck Collaborative Delivery System
## Project Overview
This repository houses the development of a **multi-agent reinforcement learning (MARL)** framework tailored for unmanned aerial vehicle (UAV) and truck collaborative delivery systems. The core objective of this project is to optimize the end-to-end delivery efficiency and enhance resource utilization in last-mile logistics scenarios through intelligent coordination between ground-based trucks and aerial UAVs.

The codebase is currently being updated.
